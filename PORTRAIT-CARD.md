# 視覺肖像卡 導讀（Colour Portrait Card）

> 這份文件回答三件事：**什麼是視覺肖像卡**、**它與五類 COLOR-TYPE 的關係**、**怎麼讀它**。
> 色彩肖像的整體想法見 [COLOR-PORTRAIT.md](COLOR-PORTRAIT.md)；五類色型見 [COLOR-TYPES.md](COLOR-TYPES.md)。

---

## 1. 什麼是視覺肖像卡？

色彩肖像有**兩種呈現**，來自同一份結構化描述（Description）：

```
describe(五構面)  →  Description  ─┬─►  phrase()  →  一句話
                                   └─►  card()    →  一張 SVG「色彩指紋卡」
```

**視覺肖像卡**就是 `card()` 的產物——把「一句話講的事」畫成一個**緊湊的圖標**。
它不是取代句子，而是句子的搭檔：**句子用文字命名，卡用圖形展示**。一眼掃過，就看到這張圖的
「配色骨架、冷暖、調性、跳出來的重點色」。出現在**明細面板**（`#detail-card`）與**完整報告**右欄各一張。

版面（由左到右四個儀表）：

```
┌──────────────────────────────────────────────────────────┐
│   ◍ 和諧環            ▬▬▬▬  溫度條          ■ 焦點色塊       │
│  (色相環+聚極點)                             FC192          │
│   〔方案名〕          ▢  明度×彩度座標        印度紅         │
└──────────────────────────────────────────────────────────┘
```

## 2. 與五類 COLOR-TYPE 的關係

**關鍵：卡不是「把五類色型畫五次」，而是它們的「蒸餾」。**

- 完整報告裡的**五條色帶**＝五類 COLOR-TYPE 的**原樣**（色族／主色／全收／分布／重點色，各一條）。
- **視覺肖像卡**＝把這五副眼鏡**跨層比較、蒸餾**後的結果（經由 `describe()` 的跨構面推理）。

所以在報告裡你會同時看到：**原始的五條**（帶）＋**蒸餾的一張**（卡）。
卡上每個儀表，都可以追溯回它「讀自哪個色型」：

| 卡上的儀表 | 讀自哪個 COLOR-TYPE | 它在回答 |
|---|---|---|
| **和諧環** | **分布**（把彩色色相聚成「極」） | 這張圖的配色骨架（單色/類比/互補/三角…） |
| **溫度條** | **分布**（暖/中/冷的真實面積） | 冷暖傾向與比例 |
| **明度×彩度座標** | **分布**（面積加權的明度、彩度）；點的顏色＝**主色** | 它坐落在色調空間的哪裡 |
| **焦點色塊** | **重點色**（顯著卻面積小的焦點）＋ Faber-Castell 命名 | 哪個顏色最跳、它最接近哪支色鉛 |

> 一句話：**色型是原始鏡片，卡是綜合判讀。**

## 3. 怎麼讀視覺肖像卡？

### ◍ 和諧環（左）——配色骨架
- **彩虹圈**＝色相環（紅在上，順時針一圈）。
- **圓點**＝這張圖的主要「色相極」；**點越大＝該色相佔的面積越大**。
- **連線/形狀**＝配色方案的幾何：
  - 1 點 → **單色**｜2 點靠近 → **類比**｜2 點相對（一條線）→ **互補**
  - 3 點成三角 → **三角**｜一點＋對面兩點 → **分裂互補**｜4 點兩組相對 → **四角**
- **環下方文字**＝方案名（varied/中性時不顯示）。
- **怎麼讀**：看幾個點、怎麼連。「兩點對拉一條線」＝互補；「三點鼎立」＝三角。（配色方案的理論與怎麼算出來，見 [HARMONY.md](HARMONY.md)。）

### ▬ 溫度條（中上）——冷暖
- 一條橫條，依面積分成 **暖（橘）｜中性（灰）｜冷（藍）** 三段。
- **怎麼讀**：藍段長＝冷調為主；橘段長＝暖；灰段長＝去飽和/中性居多。

### ▢ 明度×彩度座標（中下）——調性位置
- 一個方框：**橫軸＝彩度**（左濁 → 右鮮）、**縱軸＝明度**（下暗 → 上亮）。
- 框內**一個點**＝這張圖坐落的位置；**點的顏色＝主色**。
- **怎麼讀**：點在**右上**＝又亮又鮮；**左下**＝又暗又濁；置中＝中間調、中等飽和。

### ■ 焦點色塊（右）——最跳的重點色
- 一塊色票＝**焦點色**（小而鮮、顯著性 ≠ 面積的那個驚喜）。
- 右側文字＝它最接近的 **Faber-Castell 色**（色號 ＋ 在地化色名，如 `FC192 印度紅`）。
- **怎麼讀**：這就是「你的眼睛會先看到」的那一點顏色，以及它對應哪支實體色鉛。
- 註：若這張圖沒有明確焦點（沒有小而鮮的驚喜），這格改顯示**主色**。

## 讀一張真卡（範例）

某張側躺人像的卡：

- **和諧環**：兩個點——藍（約 200°）與暖紅（約 20°）**對拉一條線** → 環下寫「**互補**」。
- **溫度條**：藍、灰、暖三段都不小 → 冷暖並存。
- **座標點**：落在中間偏中性 → 中間調、中等飽和。
- **焦點色塊**：暗玫瑰紅，標 **FC192 印度紅**。

合起來讀，正好是那句話：

> 互補配色、看似中性，實則藍以約 35% 面積悄悄主導、色調中等飽和、中間調、焦點是一抹小而鮮的印度紅（≈FC192）。

**卡與句子講的是同一件事——一個用看的，一個用讀的。**

## 一些細節

- **顏色恆為實色**（SVG `fill`）→ 列印/存 PDF 必印；**文字用 `currentColor`** → 隨深/淺主題自動變色；框線用半透明灰（雙主題通用）。
- **純函式產出**：`ColorPortraitLib.card(desc, t, { fcName })` 回一段 SVG 字串、不碰 DOM——明細與報告共用同一份。
