# 和諧配色（Colour Harmony）——理論與導出

> 這份文件回答兩件事：**什麼是和諧理論**、**COLOR-TYPE 怎麼把它導出來**。
> 它是色彩肖像的一個構件——判讀結果會寫進句子（「互補配色」）也畫進視覺肖像卡的「和諧環」。
> 相關：五類色型 [COLOR-TYPES.md](COLOR-TYPES.md)｜色彩肖像 [COLOR-PORTRAIT.md](COLOR-PORTRAIT.md)｜卡導讀 [PORTRAIT-CARD.md](PORTRAIT-CARD.md)。

---

## 1. 什麼是和諧理論？

色彩理論裡，**顏色在「色相環」上的角度關係**會產生幾種公認好看的組合。這些「配色方案」
傳統上是**給設計者的規則**（該怎麼配）；在這裡我們反過來用它——**描述一張圖已經是哪種配色**。

色相環＝把色相 0–360° 排成一圈（紅→橙→黃→綠→青→藍→紫→洋紅→回到紅）。六種方案：

| 方案 | 色環上的幾何 | 直覺 | 例 |
|---|---|---|---|
| **單色 Monochrome** | 單一色相（一個點） | 同一個顏色的濃淡明暗 | 全藍調 |
| **類比 Analogous** | 相鄰、擠在一段窄弧內 | 隔壁鄰居，溫和 | 藍–青–綠 |
| **互補 Complementary** | 兩色相對，約 180° | 最強對比 | 藍 ↔ 橙 |
| **分裂互補 Split-comp.** | 一個基色 ＋ 其對面**兩側** | 對比但較柔和 | 藍 ＋ 橙紅、黃橙 |
| **三角 Triadic** | 三色各差約 120°（鼎立） | 均衡而活潑 | 紅–黃–藍 |
| **四角 Tetradic** | 兩組互補對（矩形/正方形） | 最豐富 | 紅綠 ＋ 橙青 |

```
   互補            三角              四角
    ●               ●               ●───●
    │              ╱ ╲              │   │
    │             ●───●             ●───●
    ●         （各120°）        （兩組相對）
```

> 重點：和諧＝**色相之間的角度關係**。所以要判它，得先知道「這張圖有哪些色相、各佔多少」。

## 2. COLOR-TYPE 怎麼導出和諧理論？

### 為什麼是「分布」這一型

和諧＝**色相 × 面積**。要誠實回答「哪些色相、各佔多少」，只有 **分布（Distribution）** 這一型合適：

- **色族／主色**：只挑 ≤12 個代表色，色相被抽樣、面積被簡化——不適合量角度關係。
- **重點色**：是「顯著性」不是面積——會誇大小而鮮的色。
- **分布**：ΔE≈5 感知分箱、**真實面積**——正是和諧要的輸入。

所以 `harmonyOf()` 吃的是**分布**的色票（每個有色相 `hue` 與面積 `ratio`）。

### 四步導出（`harmonyOf(distribution)`）

```
分布色票 ─①過濾彩色─► ②色相聚極 ─③取顯著極─► ④幾何分類 ─► 配色方案
```

**① 過濾彩色**：丟掉近中性（低彩度）色——它們沒有穩定色相，不參與配色。

**② 色相聚極（circular leader clustering）**：把剩下的色相在色環上聚成「極（pole）」。
   - 種子＝面積最大的色；其餘色若與某極**色相差 < 35°** 就併進去（面積加權，用向量和維持圓形平均），否則自成新極。
   - 為什麼要聚：原始色相很碎（漸層、雜訊），先聚成幾個「主要的聲音」才量得準。
   - **35° 這個門檻，正是「單色 vs 類比」的分界**——差不到 35° 的色相會被視為同一個色相區。

**③ 取顯著極**：只留**面積 ≥ 7%** 的極。一個色相要佔夠面積，才算得上是配色的一員（濾掉零星色）。

**④ 幾何分類**：依**顯著極的數量 n ＋ 彼此的圓形角距**判方案：

| n（極數） | 判斷 | 方案 |
|:--:|---|---|
| 0 | 沒有彩色極 | **neutral**（中性，靜默） |
| 1 | 單一色相 | **單色** |
| 2 | 角距 ≤ 40° | **類比** |
|   | 角距 ≥ 145° | **互補** |
|   | 其間 | varied |
| 3 | 全部擠在 ≤ 90° 弧內 | **類比** |
|   | 三間距都 ≈ 120°（±35） | **三角** |
|   | 一小間距 ＋ 兩個 ≥120° 且相近 | **分裂互補** |
|   | 其餘 | varied |
| 4 | 能配成兩組互補對（各 180°±32） | **四角** |
|   | 擠在 ≤ 90° 弧內 | **類比** |
|   | 其餘 | varied |
| ≥5 | — | varied |

> `varied`／`neutral` 代表「沒有明確方案」——句子與卡上都**不硬套**，靜默帶過。

### 描述，而非規範

傳統色彩理論說「你**該**這樣配」；這裡是反過來——量出「這張圖**已經**是哪種配」。
它不評好壞、不建議，只是把畫面的配色骨架讀出來，成為色彩肖像的一部分。

### 導出之後去哪

- **句子**：明確方案時加一句「互補配色／類比配色…」；跨色環方案（互補/分裂互補/三角/四角）本身即含暖冷對比，故與「暖冷對比」clause 去重。
- **視覺肖像卡的和諧環**：把②的**極**畫成色環上的點（大小＝面積）、④的**幾何**畫成連線（一條＝互補、三角形＝三角…）、方案名寫在環下。怎麼讀見 [PORTRAIT-CARD.md](PORTRAIT-CARD.md)。

## 一個實例

某張側躺人像：分布裡的彩色色相聚成**兩極**——藍（≈200°，寢具背景）與暖紅（≈20°，膚色/唇）。
兩極角距約 175° ≥ 145° → **互補**。於是句子開頭是「互補配色」，卡上兩點對拉一條線、環下寫「互補」。
