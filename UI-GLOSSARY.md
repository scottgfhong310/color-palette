# color-palette 介面元素命名表（UI Glossary）

> 這份文件替 **每個畫面上的每個元素** 訂一個一致的名字，用途有二：
> **①介紹 app** 時大家講同一套詞；**②和 Claude 對話**時能精準指到某個元素（例：「明細的**潤稿鈕**」＝ `#detail-polish`）。
> 每項格式：**中文名稱** ｜ 一句說明 ｜ `程式錨點`（id 用 `#`、class 用 `.`）。
> 概念（色型/肖像/遮罩演算法…）不在這裡，各有專文：[COLOR-TYPES](COLOR-TYPES.md)／[COLOR-PORTRAIT](COLOR-PORTRAIT.md)／[POLISH](POLISH.md) 等。

---

## 0. 三個畫面與動線

```
首頁（Gallery）  ──點一張卡──►  明細（Detail）  ──點原圖──►  燈箱（Lightbox）
   色票櫃全覽                    一張圖的細節                   細看原圖＋取色＋遮罩
```

| 畫面 | 一句話 | 根容器 |
|---|---|---|
| **首頁 / 色票櫃**（Gallery） | 整櫃圖的色票牆，依色系分群瀏覽 | `#gallery` |
| **明細**（Detail） | 單張圖的色彩肖像＋五種顏色視圖＋匯出 | `#detail-modal` |
| **燈箱**（Lightbox） | 放大細看原圖、滴管取色、產生遮罩 | `#lightbox` |

## 1. 首頁（Gallery）

```
┌───────────────────────────────────────────────┐   ┌─右側工具列─┐
│ 篩選列（依色彩肖像）                            │   │ ⬆ 上傳     │
│ 圖庫肖像：83 張 · 整體偏暖 · 單色為主 · 近期偏暖 │   │ ✨ 萃取法   │
│ ┌色系標頭「紅 12」──────────────────────────┐ │   │ ↻ 重新分析  │
│ │ [卡] [卡] [卡] …                          │ │   │ ▦ 密度     │
│ └──────────────────────────────────────────┘ │   │ ⛃ 篩選     │
│ ┌色系標頭「藍 20」─────────────────────────┐  │   │ ☾ 主題     │
│ │ [卡] [卡] …                              │  │   │ 文 語言    │
│ └─────────────────────────────────────────┘  │   │ 🗑 清空     │
└───────────────────────────────────────────────┘   └───────────┘
 色軌（左側）                                          
```

### 版面
| 名稱 | 說明 | 錨點 |
|---|---|---|
| **空狀態** | 沒有圖時的上傳提示（可點、可全頁拖拉） | `#empty-state` |
| **拖拉覆蓋層** | 拖檔到頁面時的全頁提示 | `#drop-overlay` |
| **篩選列** | 依色彩肖像標籤篩選（溫度／原型／和諧／明度）；未啟用時整列隱藏 | `#filter-bar` |
| ⤷ **篩選 chip** | 篩選列裡的單一標籤鈕 | `.filter-chip` |
| ⤷ **清除篩選** | 清掉所有篩選 | `.filter-clear` |
| **圖庫肖像** | 頂端一行「關係與時間」摘要（尊重篩選） | `#collection-portrait` |
| **色票櫃 / gallery** | 卡片區，依色系分群 | `#gallery` |
| **跳轉色軌** | 左側色點,點一下捲到該色系 | `#jump-rail`（點＝`.jump-dot`） |

| 名稱 | 說明 | 錨點 |

| 名稱 | 說明 | 錨點 |

| **卡片** | 一張圖一張卡（點開明細） | `.pcard` |
| ⤷ **縮圖** | 舒適密度時的圖縮圖 | `.pcard-thumb` |
| ⤷ **卡片色票列** | 色票牆密度時的色塊列 | `.swatches` ＞ `.swatch` |
| ⤷ **萃取法徽章** | 標 `MEDIAN`／`FREQUENCY`（＝這張的指紋算法） | `.pcard-method` |
| ⤷ **卡片檔名** | 圖的檔名 | `.pcard-name` |

### 右側工具列
| 名稱 | 說明 | 錨點 |
|---|---|---|
| **右側工具列** | 8 顆功能鈕的直排 | `.side-tools` |
| ⤷ **上傳** | 選檔上傳 | `#setting-upload` |
| ⤷ **萃取法** | 切色族(median)／頻率(frequency) | `#setting-method` |
| ⤷ **重新分析** | 以目前萃取法重算全部指紋 | `#setting-reanalyze` |
| ⤷ **瀏覽密度** | 縮圖 ↔ 色票牆 | `#setting-density` |
| ⤷ **篩選** | 開/關篩選列 | `#setting-filter` |
| ⤷ **主題** | light ↔ dark | `#setting-mode` |
| ⤷ **語言** | zh-Hant / en / ja 循環 | `#setting-lang` |
| ⤷ **清空** | 刪光資料夾（`confirm`） | `#setting-clear` |

### 共用
| 名稱 | 說明 | 錨點 |
|---|---|---|
| **載入動畫** | 上傳/分析時的轉圈 | `#loading` |
| **Toast** | 右下角短訊息（複製成功、切換提示…） | Materialize toast |

## 2. 明細（Detail）

點一張卡開啟。

```
┌ 明細 Modal ────────────────────────────────┐
│ [原圖]  檔名                                 │  ← 圖頭
│         副標（視圖 · 色數 · 大小）            │
│ ▏色彩肖像句 …………………………………… [✨]   │  ← 肖像列（句＋潤稿鈕）
│ [視覺肖像卡 SVG]                             │
│ [色族][主色][全收][分布][重點色]  ⧉ 📖 Σ    │  ← 分頁 ＋ 動作
│ ■ #hex   佔比%   ≈ FC### 名 ΔE              │  ← 明細清單（逐色）
│ ■ …                                         │
│─────────────────────────────────────────────│
│ 🗑刪除            重新分析          關閉      │  ← 頁尾
└─────────────────────────────────────────────┘
```

| 名稱 | 說明 | 錨點 |
|---|---|---|
| **明細 Modal** | 整個彈窗 | `#detail-modal` |
| **圖頭** | 頂部原圖＋檔名區 | `.detail-head` |
| ⤷ **明細原圖** | 縮圖（點它開燈箱） | `#detail-image` |
| ⤷ **明細檔名** | 檔名 | `#detail-name` |
| ⤷ **副標** | 視圖 · 色數 · 檔案大小 | `#detail-sub` |
| **色彩肖像列** | 肖像句＋潤稿鈕同一列 | `.detail-portrait-row` |
| ⤷ **色彩肖像句** | 一句色彩描述（決定論） | `#detail-portrait` |
| ⤷ **潤稿鈕** | ✨ 用 LLM 潤飾那句（見 [POLISH](POLISH.md)） | `#detail-polish` |
| **視覺肖像卡** | SVG 色彩指紋卡（見 [PORTRAIT-CARD](PORTRAIT-CARD.md)） | `#detail-card` |
| **萃取視圖分頁** | 五種顏色視圖切換（見 [COLOR-TYPES](COLOR-TYPES.md)） | `#detail-tabs` |
| ⤷ **分頁：色族/主色/全收/分布/重點色** | 各一分頁 | `.detail-tab[data-view=…]` |
| **複製全部** | ⧉ 複製目前視圖所有色碼 | `#detail-copyall` |
| **存 .md** | 📖 存單構面色票 `.md`（交 markdown-library） | `#detail-md` |
| **完整報告** | Σ 存五構面 A4 分頁 `.md` | `#detail-report` |
| **明細清單** | 逐色列表 | `#detail-list` |
| ⤷ **色塊** | 每列左邊的顏色方塊 | `.detail-chip` |
| ⤷ **佔比** | 該色比例（意義依視圖，見色型文件） | `.detail-ratio` |
| ⤷ **最接近 FC** | ≈ Faber-Castell 色鉛＋ΔE（＋2 替代色） | `.fc-near` |
| **刪除** | 刪這張圖 | `#detail-delete` |
| **重新分析** | 用目前萃取法改寫這張的指紋 | `#detail-reanalyze` |

## 3. 燈箱（Lightbox）

在明細點原圖開啟。左＝舞台，右＝色票側欄，底＝資訊列。

```
┌ 燈箱 ─────────────────────────┬─色票側欄──────┐
│                               │ 取色讀值 ■#hex✕│  ← 釘住/滑過取色
│         [ 原圖＋遮罩 ]         │ [色票|分布|重點]│  ← 遮罩基準
│          （可縮放平移）        │ ⬤紅 ⬤藍 ⬤中性 │  ← 色系遮罩 chip
│                               │ ■ #hex % ≈FC   │  ← 色票列（逐色）
│                               │ ■ …            │
├───────────────────────────────┴────────────────┤
│ 檔名  縮放%   滾輪縮放·點圖釘色·點色系/色票定位  ⛶ ✕│  ← 底部列（含 fit）
└─────────────────────────────────────────────────┘
        （游標旁：取色放大鏡）
```

### 舞台與圖
| 名稱 | 說明 | 錨點 |
|---|---|---|
| **燈箱** | 整個全螢幕檢視 | `#lightbox` |
| **舞台** | 承載圖、負責縮放平移的區域 | `#lightbox-stage` |
| **原圖** | 被看的那張圖 | `#lightbox-img` |
| **遮罩** | 「選色產生遮罩」的暗遮疊層（與圖同框對位） | `#lightbox-mask` |

### 色票側欄
| 名稱 | 說明 | 錨點 |
|---|---|---|
| **色票側欄** | 右側整條 | `#lightbox-palette` |
| **取色讀值** | 目前取到/釘住的色：色塊＋hex＋≈FC＋✕（見取色/釘住動詞） | `#lightbox-pick`（✕＝`.lightbox-pick-unpin`） |
| **遮罩基準** | 色票列依哪個 COLOR-TYPE：色票／分布／重點色 | `#lightbox-basis`（`.lb-basis-btn`） |
| **色系遮罩 chip** | 點一個色系＝整族像素定位 | `#lightbox-families`（`.lb-family`） |
| **燈箱色票列** | 逐色列表（色塊＋hex＋%＋≈FC），點一色＝單色定位 | `#lightbox-swatches`（`.lb-swatch`） |

### 底部列
| 名稱 | 說明 | 錨點 |
|---|---|---|
| **底部列** | 檔名/縮放/提示/按鈕 | `.lightbox-bar` |
| ⤷ **燈箱檔名** | 檔名 | `#lightbox-name` |
| ⤷ **縮放指示** | 目前縮放 % | `#lightbox-zoom` |
| ⤷ **操作提示** | 一行手勢提示 | `.lightbox-hint` |
| ⤷ **符合視窗鈕** | ⛶ 縮放回完整 fit | `#lightbox-fit` |
| ⤷ **關閉** | ✕ 關燈箱 | `#lightbox-close` |
| **取色放大鏡** | 跟游標的像素放大鏡＋十字＋hex | `#picker-loupe` |

## 4. 跨頁的「動作」動詞（對話時用同一組）

| 動詞 | 意思 | 在哪 |
|---|---|---|
| **上傳 / 分析** | 加圖、算出色票指紋 | 首頁 |
| **切萃取法** | 色族(median) ↔ 頻率(frequency) | 首頁工具列 |
| **切視圖** | 明細的色族/主色/全收/分布/重點色 | 明細分頁 |
| **潤稿** | 用 LLM 把肖像句改寫更漂亮 | 明細 ✨ |
| **篩選** | 依色彩肖像標籤過濾色票櫃 | 首頁篩選列 |
| **取色（滴管）** | 滑過原圖讀某點的色＋最接近 FC | 燈箱 |
| **釘住** | 點原圖固定住那點的色（可複製） | 燈箱 |
| **定位 / 產生遮罩** | 讓某色/某色系/某基準的像素透出、其餘變暗 | 燈箱 |
| **切基準** | 遮罩依 色票／分布／重點色 | 燈箱遮罩基準 |
| **符合視窗（fit）** | 把原圖縮放回完整可見 | 燈箱 ⛶ |

## 命名慣例（給新元素）

- **畫面容器**用畫面名：`#gallery` / `#detail-modal` / `#lightbox`。
- **同一畫面內**的元素前綴一致：明細＝`detail-*`、燈箱＝`lightbox-*`／`lb-*`、卡片＝`pcard-*`、色系＝`hue-*`／`lb-family`、篩選＝`filter-*`。
- **中文名**以「這個東西是什麼」命名（色系標頭、取色讀值、遮罩基準…），對話時中文名與程式錨點皆可，能對上即可。
